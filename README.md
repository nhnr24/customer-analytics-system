# Customer Analytics & Segmentation System

## Overview

This project analyzes customer data and performs segmentation based on spending behavior and income levels to generate business insights.

## Features

* Data cleaning and preprocessing
* Customer segmentation (High, Medium, Low value)
* Income-based grouping
* City-wise analysis
* Data visualization using charts
* Exporting processed results

## Key Insights

* High-value customers show higher spending
* High-income group spends more
* Hyderabad has highest spending customers

## Workflow

1. Load data
2. Clean data
3. Feature engineering
4. Segmentation
5. Analysis
6. Visualization

## Tools Used

Python, Pandas, Matplotlib

* Identified top spending city
* Identified dominant customer segment
